# Integrate NX with dotnetcore

```bash
npm i --save-dev @nx-dotnet/core
npx nx g @nx-dotnet/core:init
npx nx g @nx-dotnet/core:app gatehub --test-template nunit --language C# --solutionFile gatehub.sln
```
