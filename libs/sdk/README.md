# @nineteensevenfour/gate-sdk

[![semantic-release-plus: angular](https://img.shields.io/badge/semantic--release-angular-e10079?logo=semantic-release)](https://github.com/semantic-release-plus/semantic-release)

This library provide a all-in-one SDK for developer who whish to create a GATE micro-frontend web application with features such as navigation, security, settings, registration, communication.

## Roadmap

Please refers to the [github project milestones](https://github.com/orgs/NineteenSevenFour/projects/17/views/4) for an overview of the roadmap.

## Contribution

Anyone can join, but this is an ongoing as free time allow kind of project I use as a sandbox to testing things out out of working hours.
